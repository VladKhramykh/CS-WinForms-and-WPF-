	ЗАМЕНА
1-я строка: Вводите исходную строку
2-я строка: Вводите значение(через запятую), какое нужно заменить и значение, на которое должна произойти замена
Должно быть два значения!
Пример2-1 строки: Старое,новое

	УДАЛЕНИЕ ПОДСТРОКИ

2-я строка: Вводите значение какое нужно удалить
Должно быть два значения во 2-й строке!
Пример 2-й строки: Удалить

	СИМВОЛ ПО ИНДЕКСУ
